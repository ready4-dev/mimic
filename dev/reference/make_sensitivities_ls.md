# Make sensitivities list

make_sensitivities_ls() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make
sensitivities list. The function returns Sensitivities (a list).

## Usage

``` r
make_sensitivities_ls(timestamp_1L_chr = "_YR1")
```

## Arguments

- timestamp_1L_chr:

  Timestamp (a character vector of length one), Default: '\_YR1'

## Value

Sensitivities (a list)
