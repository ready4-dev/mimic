# Make worker types

make_worker_types() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make worker
types. The function returns Worker types (a character vector).

## Usage

``` r
make_worker_types(type_1L_chr = c("all", "medical"))
```

## Arguments

- type_1L_chr:

  Type (a character vector of length one), Default: c("all", "medical")

## Value

Worker types (a character vector)
