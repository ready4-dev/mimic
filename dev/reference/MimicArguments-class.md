# MimicArguments

Method argument data.

## Slots

- `iterations_1L_lgl`:

  Iterations (a logical vector of length one)

- `derive_ls`:

  Derive (a list)

- `models_ls`:

  Models (a list)

- `x_MimicDerivations`:

  (an instance of the MimicDerivations class)

- `dissemination_1L_chr`:

  Dissemination (a character vector of length one)
