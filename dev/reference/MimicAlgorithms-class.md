# MimicAlgorithms

The core set of functions that define simulation behaviour.

## Slots

- `main_ls`:

  Main (a list)

- `processing_ls`:

  Processing (a list)

- `sensitivities_ls`:

  Sensitivities (a list)

- `transformations_ls`:

  Transformations (a list)

- `dissemination_1L_chr`:

  Dissemination (a character vector of length one)
