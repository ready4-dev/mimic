# Update current event

update_current_event() is an Update function that edits an object, while
preserving core object attributes. Specifically, this function
implements an algorithm to update current event. The function is called
for its side effects and does not return a value.

## Usage

``` r
update_current_event(X_Ready4useDyad)
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

## Value

X (A dataset and data dictionary pair.)
