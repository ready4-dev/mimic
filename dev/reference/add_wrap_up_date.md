# Add wrap up date

add_wrap_up_date() is an Add function that updates an object by adding
new values to new or empty fields. Specifically, this function
implements an algorithm to add wrap up date. The function is called for
its side effects and does not return a value.

## Usage

``` r
add_wrap_up_date(X_Ready4useDyad)
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

## Value

X (A dataset and data dictionary pair.)
