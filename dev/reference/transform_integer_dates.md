# Transform integer dates

transform_integer_dates() is a Transform function that edits an object
in such a way that core object attributes - e.g. shape, dimensions,
elements, type - are altered. Specifically, this function implements an
algorithm to transform integer dates. The function returns Dates (a date
vector).

## Usage

``` r
transform_integer_dates(dates_int)
```

## Arguments

- dates_int:

  Dates (an integer vector)

## Value

Dates (a date vector)
