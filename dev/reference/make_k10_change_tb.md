# Make K10 change tibble

make_k10_change_tb() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make k10 change
tibble. The function returns K10 change (a tibble).

## Usage

``` r
make_k10_change_tb(population_k10_tb)
```

## Arguments

- population_k10_tb:

  Population K10 (a tibble)

## Value

K10 change (a tibble)
