# Update population classes

update_population_classes() is an Update function that edits an object,
while preserving core object attributes. Specifically, this function
implements an algorithm to update population classes. The function is
called for its side effects and does not return a value.

## Usage

``` r
update_population_classes(X_Ready4useDyad, tfmn_ls = NULL)
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

- tfmn_ls:

  Transformation (a list), Default: NULL

## Value

X (A dataset and data dictionary pair.)
