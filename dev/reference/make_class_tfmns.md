# Make class transformations

make_class_tfmns() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make class
transformations. The function returns Transformations (a list).

## Usage

``` r
make_class_tfmns(force_1L_lgl = FALSE)
```

## Arguments

- force_1L_lgl:

  Force (a logical vector of length one), Default: FALSE

## Value

Transformations (a list)
