# Update K10 event schedule

update_k10_event_schedule() is an Update function that edits an object,
while preserving core object attributes. Specifically, this function
implements an algorithm to update k10 event schedule. The function is
called for its side effects and does not return a value.

## Usage

``` r
update_k10_event_schedule(X_Ready4useDyad, type_1L_chr = c("Model", "Table"))
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

- type_1L_chr:

  Type (a character vector of length one), Default: c("Model", "Table")

## Value

X (A dataset and data dictionary pair.)
