# Make economic results

make_economic_results() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make economic
results. The function returns Economic results (a list).

## Usage

``` r
make_economic_results(sim_results_ls, intervention_1L_chr, utilities_chr)
```

## Arguments

- sim_results_ls:

  Sim results (a list)

- intervention_1L_chr:

  Intervention (a character vector of length one)

- utilities_chr:

  Utilities (a character vector)

## Value

Economic results (a list)
