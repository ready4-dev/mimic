# Make project joiners dataset

make_project_joiners_ds() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make
project joiners dataset. The function returns Data (an output object of
multiple potential types).

## Usage

``` r
make_project_joiners_ds(processed_ls, as_dyad_1L_lgl = T)
```

## Arguments

- processed_ls:

  Processed (a list)

- as_dyad_1L_lgl:

  As dyad (a logical vector of length one), Default: T

## Value

Data (an output object of multiple potential types)
