# Make Minimum Dataset episodes tibble

make_mds_episodes_tb() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make minimum
dataset episodes tibble. The function returns Episodes (a tibble).

## Usage

``` r
make_mds_episodes_tb(raw_mds_data_ls, processed_ls)
```

## Arguments

- raw_mds_data_ls:

  Raw Minimum Dataset data (a list)

- processed_ls:

  Processed (a list)

## Value

Episodes (a tibble)
