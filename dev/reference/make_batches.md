# Make batches

make_batches() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make batches. The
function returns Batches (a list).

## Usage

``` r
make_batches(n_1L_int = 50, of_1L_int = 20)
```

## Arguments

- n_1L_int:

  N (an integer vector of length one), Default: 50

- of_1L_int:

  Of (an integer vector of length one), Default: 20

## Value

Batches (a list)
