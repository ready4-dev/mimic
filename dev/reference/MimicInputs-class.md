# MimicInputs

Model input data (lookup tables, regression models, population and
parameters).

## Slots

- `lookups_ls`:

  Lookups (a list)

- `models_ls`:

  Models (a list)

- `x_Ready4useDyad`:

  (an instance of the Ready4useDyad class)

- `y_Ready4useDyad`:

  (an instance of the Ready4useDyad class)

- `dissemination_1L_chr`:

  Dissemination (a character vector of length one)
