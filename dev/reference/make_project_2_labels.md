# Make project 2 labels

make_project_2_labels() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make project 2
labels. The function returns Labels (a list).

## Usage

``` r
make_project_2_labels(type_1L_chr = c("regression", "simulation"))
```

## Arguments

- type_1L_chr:

  Type (a character vector of length one), Default: c("regression",
  "simulation")

## Value

Labels (a list)
