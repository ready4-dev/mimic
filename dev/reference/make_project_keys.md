# Make project keys

make_project_keys() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make project
keys. The function returns Keys (a character vector).

## Usage

``` r
make_project_keys(type_1L_chr = c("micro", "ts"))
```

## Arguments

- type_1L_chr:

  Type (a character vector of length one), Default: c("micro", "ts")

## Value

Keys (a character vector)
