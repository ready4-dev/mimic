# Update arguments list

update_arguments_ls() is an Update function that edits an object, while
preserving core object attributes. Specifically, this function
implements an algorithm to update arguments list. The function returns
Arguments (a list).

## Usage

``` r
update_arguments_ls(args_ls, function_fn)
```

## Arguments

- args_ls:

  Arguments (a list)

- function_fn:

  Function (a function)

## Value

Arguments (a list)
