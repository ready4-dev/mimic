# Update arguments list

update_arguments_ls() is an Update function that edits an object, while
preserving core object attributes. Specifically, this function
implements an algorithm to update arguments list. The function returns
Arguments (a list).

## Usage

``` r
update_arguments_ls(args_ls, function_fn, allowed_chr = character(0))
```

## Arguments

- args_ls:

  Arguments (a list)

- function_fn:

  Function (a function)

- allowed_chr:

  Allowed (a character vector), Default: character(0)

## Value

Arguments (a list)
