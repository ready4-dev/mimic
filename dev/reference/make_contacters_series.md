# Make contacters series

make_contacters_series() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make contacters
series. The function is called for its side effects and does not return
a value.

## Usage

``` r
make_contacters_series(model_data_ls)
```

## Arguments

- model_data_ls:

  Model data (a list)

## Value

X (A dataset and data dictionary pair.)
