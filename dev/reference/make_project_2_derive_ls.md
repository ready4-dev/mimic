# Make project 2 derive list

make_project_2_derive_ls() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make
project 2 derive list. The function returns Derive (a list).

## Usage

``` r
make_project_2_derive_ls(
  function_fn = NULL,
  discard_chr = character(0),
  keep_chr = character(0)
)
```

## Arguments

- function_fn:

  Function (a function), Default: NULL

- discard_chr:

  Discard (a character vector), Default: character(0)

- keep_chr:

  Keep (a character vector), Default: character(0)

## Value

Derive (a list)
