# Make Australia price years

make_aus_price_years() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make australia
price years. The function returns Price indices (a double vector).

## Usage

``` r
make_aus_price_years(start_1L_chr = "2021-22", end_1L_chr = "2024-25")
```

## Arguments

- start_1L_chr:

  Start (a character vector of length one), Default: '2021-22'

- end_1L_chr:

  End (a character vector of length one), Default: '2024-25'

## Value

Price indices (a double vector)
