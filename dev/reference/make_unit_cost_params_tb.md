# Make unit cost parameters tibble

make_unit_cost_params_tb() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make unit
cost parameters tibble. The function returns Unit cost parameters (a
tibble).

## Usage

``` r
make_unit_cost_params_tb(cost_per_mins_tb, new_ls = NULL)
```

## Arguments

- cost_per_mins_tb:

  Cost per minimums (a tibble)

- new_ls:

  New (a list), Default: NULL

## Value

Unit cost parameters (a tibble)
