# Make project metrics

make_project_metrics() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make project
metrics. The function returns Metrics (a character vector).

## Usage

``` r
make_project_metrics(unit_costs_tb)
```

## Arguments

- unit_costs_tb:

  Unit costs (a tibble)

## Value

Metrics (a character vector)
