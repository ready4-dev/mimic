# Update next event

update_next_event() is an Update function that edits an object, while
preserving core object attributes. Specifically, this function
implements an algorithm to update next event. The function is called for
its side effects and does not return a value.

## Usage

``` r
update_next_event(X_Ready4useDyad)
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

## Value

X (A dataset and data dictionary pair.)
