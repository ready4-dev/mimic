# Add project offset logic

add_project_offset_logic() is an Add function that updates an object by
adding new values to new or empty fields. Specifically, this function
implements an algorithm to add project offset logic. The function
returns Data (a tibble).

## Usage

``` r
add_project_offset_logic(data_tb)
```

## Arguments

- data_tb:

  Data (a tibble)

## Value

Data (a tibble)
