# MimicPopulation

Model dynamic population data container (specifying population subgroups
who are active and eligible for the next model event, who are
temporariliy inactive but are still in the model or who have exited the
model).

## Slots

- `x_MimicActive`:

  (an instance of the MimicActive class)

- `y_Ready4useDyad`:

  (an instance of the Ready4useDyad class)

- `z_Ready4useDyad`:

  (an instance of the Ready4useDyad class)

- `dissemination_1L_chr`:

  Dissemination (a character vector of length one)
