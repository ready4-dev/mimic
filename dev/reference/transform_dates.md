# Transform dates

transform_dates() is a Transform function that edits an object in such a
way that core object attributes - e.g. shape, dimensions, elements,
type - are altered. Specifically, this function implements an algorithm
to transform dates. The function returns Dates (a date vector).

## Usage

``` r
transform_dates(dates_chr)
```

## Arguments

- dates_chr:

  Dates (a character vector)

## Value

Dates (a date vector)
