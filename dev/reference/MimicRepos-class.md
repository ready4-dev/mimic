# MimicRepos

Local and remote repositories for model input and output data.

## Slots

- `batch_to_1L_chr`:

  Batch to (a character vector of length one)

- `draw_to_1L_chr`:

  Draw to (a character vector of length one)

- `divider_1L_chr`:

  Divider (a character vector of length one)

- `outp_data_dir_1L_chr`:

  Output data directory (a character vector of length one)

- `path_to_keys_1L_chr`:

  Path to keys (a character vector of length one)

- `path_to_output_1L_chr`:

  Path to output (a character vector of length one)

- `path_to_param_data_1L_chr`:

  Path to parameter data (a character vector of length one)

- `path_to_private_1L_chr`:

  Path to private (a character vector of length one)

- `processed_dir_1L_chr`:

  Processed directory (a character vector of length one)

- `r_dir_1L_chr`:

  R directory (a character vector of length one)

- `raw_dir_1L_chr`:

  Raw directory (a character vector of length one)

- `reports_dir_1L_chr`:

  Reports directory (a character vector of length one)

- `x_Ready4useRepos`:

  (an instance of the Ready4useRepos class)

- `dissemination_1L_chr`:

  Dissemination (a character vector of length one)
