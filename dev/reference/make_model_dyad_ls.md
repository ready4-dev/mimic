# Make model dyad list

make_model_dyad_ls() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make model dyad
list. The function returns Model dyad (a list).

## Usage

``` r
make_model_dyad_ls(
  X_Ready4useDyad = ready4use::Ready4useDyad(),
  Y_Ready4useDyad = ready4use::Ready4useDyad()
)
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION, Default: ready4use::Ready4useDyad()

- Y_Ready4useDyad:

  PARAM_DESCRIPTION, Default: ready4use::Ready4useDyad()

## Value

Model dyad (a list)
