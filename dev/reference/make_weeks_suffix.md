# Make weeks suffix

make_weeks_suffix() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make weeks
suffix. The function returns Suffix (a character vector of length one).

## Usage

``` r
make_weeks_suffix(
  X_Ready4useDyad,
  adjustment_1L_dbl = 0,
  follow_up_1L_int = integer(0)
)
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

- adjustment_1L_dbl:

  Adjustment (a double vector of length one), Default: 0

- follow_up_1L_int:

  Follow up (an integer vector of length one), Default: integer(0)

## Value

Suffix (a character vector of length one)
