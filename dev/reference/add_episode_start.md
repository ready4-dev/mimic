# Add episode start

add_episode_start() is an Add function that updates an object by adding
new values to new or empty fields. Specifically, this function
implements an algorithm to add episode start. The function is called for
its side effects and does not return a value.

## Usage

``` r
add_episode_start(X_Ready4useDyad)
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

## Value

X (A dataset and data dictionary pair.)
