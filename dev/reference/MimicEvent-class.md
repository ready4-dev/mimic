# MimicEvent

## Slots

- `x_MimicEligible`:

  (an instance of the MimicEligible class)

- `x_MimicSchedule`:

  (an instance of the MimicSchedule class)

- `x_MimicTrigger`:

  (an instance of the MimicTrigger class)

- `dissemination_1L_chr`:

  Dissemination (a character vector of length one)
