# Make project Assessment of Quality of Life Six Dimension models

make_project_aqol6d_mdls() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make
project assessment of quality of life six dimension models. The function
returns Assessment of Quality of Life Six Dimension (a list).

## Usage

``` r
make_project_aqol6d_mdls(X_Ready4useDyad)
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

## Value

Assessment of Quality of Life Six Dimension (a list)
