# Add activity

add_activity() is an Add function that updates an object by adding new
values to new or empty fields. Specifically, this function implements an
algorithm to add activity. The function returns Data (a tibble).

## Usage

``` r
add_activity(data_tb, neither_1L_chr = "ERROR")
```

## Arguments

- data_tb:

  Data (a tibble)

- neither_1L_chr:

  Neither (a character vector of length one), Default: 'ERROR'

## Value

Data (a tibble)
