# Make enhanced pool

make_enhanced_pool() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make enhanced
pool. The function returns Pooled (a list).

## Usage

``` r
make_enhanced_pool(pooled_fits_ls, arguments_ls)
```

## Arguments

- pooled_fits_ls:

  Pooled fits (a list)

- arguments_ls:

  Arguments (a list)

## Value

Pooled (a list)
