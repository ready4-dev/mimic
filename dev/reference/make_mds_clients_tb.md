# Make Minimum Dataset clients tibble

make_mds_clients_tb() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make minimum
dataset clients tibble. The function returns a Minimum Dataset clients
(a tibble).

## Usage

``` r
make_mds_clients_tb(raw_mds_data_ls)
```

## Arguments

- raw_mds_data_ls:

  Raw Minimum Dataset data (a list)

## Value

a Minimum Dataset clients (a tibble)
