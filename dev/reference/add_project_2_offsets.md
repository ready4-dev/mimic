# Add project 2 offsets

add_project_2_offsets() is an Add function that updates an object by
adding new values to new or empty fields. Specifically, this function
implements an algorithm to add project 2 offsets. The function is called
for its side effects and does not return a value.

## Usage

``` r
add_project_2_offsets(X_Ready4useDyad, arms_for_offsets_chr = character(0))
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

- arms_for_offsets_chr:

  Arms for offsets (a character vector), Default: character(0)

## Value

X (A dataset and data dictionary pair.)
