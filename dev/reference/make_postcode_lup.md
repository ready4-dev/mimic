# Make postcode lookup table

make_postcode_lup() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make postcode
lookup table. The function returns Postcode (a lookup table).

## Usage

``` r
make_postcode_lup(url_1L_chr = character(0))
```

## Arguments

- url_1L_chr:

  Url (a character vector of length one), Default: character(0)

## Value

Postcode (a lookup table)
