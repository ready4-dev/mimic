# Make project 2 theme function

make_project_2_theme_fn() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make
project 2 theme function. The function returns Plot (a function).

## Usage

``` r
make_project_2_theme_fn(output_1L_chr = c("Word", "PDF", "HTML"))
```

## Arguments

- output_1L_chr:

  Output (a character vector of length one), Default: c("Word", "PDF",
  "HTML")

## Value

Plot (a function)
