# Update project 2 parameter names

update_project_2_param_names() is an Update function that edits an
object, while preserving core object attributes. Specifically, this
function implements an algorithm to update project 2 parameter names.
The function returns Parameters (a tibble).

## Usage

``` r
update_project_2_param_names(params_tb, intervention_1L_chr)
```

## Arguments

- params_tb:

  Parameters (a tibble)

- intervention_1L_chr:

  Intervention (a character vector of length one)

## Value

Parameters (a tibble)
