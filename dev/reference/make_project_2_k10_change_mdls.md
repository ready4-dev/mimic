# Make project 2 K10 change models

make_project_2_k10_change_mdls() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make
project 2 k10 change models. The function returns K10 (a list).

## Usage

``` r
make_project_2_k10_change_mdls(
  X_Ready4useDyad,
  type_1L_chr = c("Intervention", "Jurisdiction")
)
```

## Arguments

- X_Ready4useDyad:

  PARAM_DESCRIPTION

- type_1L_chr:

  Type (a character vector of length one), Default: c("Intervention",
  "Jurisdiction")

## Value

K10 (a list)
