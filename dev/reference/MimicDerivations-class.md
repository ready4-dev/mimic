# MimicDerivations

Data for deriving method arguments.

## Slots

- `method_1L_chr`:

  Method (a character vector of length one)

- `args_env_ls`:

  Arguments (a list of environments)

- `args_fixed_ls`:

  Arguments fixed (a list)

- `dissemination_1L_chr`:

  Dissemination (a character vector of length one)
