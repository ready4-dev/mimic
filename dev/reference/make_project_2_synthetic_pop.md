# Make project 2 synthetic pop

make_project_2_synthetic_pop() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make
project 2 synthetic pop. The function returns Population (a list).

## Usage

``` r
make_project_2_synthetic_pop(model_data_ls, size_1L_int = 1000L)
```

## Arguments

- model_data_ls:

  Model data (a list)

- size_1L_int:

  Size (an integer vector of length one), Default: 1000

## Value

Population (a list)
