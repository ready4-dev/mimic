# MimicUtility

Utility algorithm data (instrument names, modules, mapping functions and
transformation functions).

## Slots

- `names_chr`:

  Names (a character vector)

- `data_ls`:

  Data (a list)

- `mapping_ls`:

  Mapping (a list)

- `transformations_ls`:

  Transformations (a list)

- `dissemination_1L_chr`:

  Dissemination (a character vector of length one)
