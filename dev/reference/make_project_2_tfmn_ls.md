# Make project 2 transformation list

make_project_2_tfmn_ls() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make project 2
transformation list. The function returns Transformation (a list).

## Usage

``` r
make_project_2_tfmn_ls(type_1L_chr = "outcome")
```

## Arguments

- type_1L_chr:

  Type (a character vector of length one), Default: 'outcome'

## Value

Transformation (a list)
