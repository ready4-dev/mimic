# Make Minimum Dataset service contacts

make_mds_service_contacts() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make
minimum dataset service contacts. The function returns Data (a tibble).

## Usage

``` r
make_mds_service_contacts(raw_mds_data_ls)
```

## Arguments

- raw_mds_data_ls:

  Raw Minimum Dataset data (a list)

## Value

Data (a tibble)
