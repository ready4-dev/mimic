# Make project 2 append list

make_project_2_append_ls() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make
project 2 append list. The function returns Append (a list).

## Usage

``` r
make_project_2_append_ls(
  arms_for_iar_adjustment_chr,
  arms_for_non_helpseeking_chr,
  never_1L_int
)
```

## Arguments

- arms_for_iar_adjustment_chr:

  Arms for Initial Assessment andeferral adjustment (a character vector)

- arms_for_non_helpseeking_chr:

  Arms for non helpseeking (a character vector)

- never_1L_int:

  Never (an integer vector of length one)

## Value

Append (a list)
