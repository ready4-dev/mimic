# Make ineligibility functions list

make_ineligibility_fns_ls() is a Make function that creates a new R
object. Specifically, this function implements an algorithm to make
ineligibility functions list. The function returns Ineligibility (a
list).

## Usage

``` r
make_ineligibility_fns_ls(post_fn = identity, pre_fn = identity)
```

## Arguments

- post_fn:

  Post (a function), Default: identity

- pre_fn:

  Pre (a function), Default: identity

## Value

Ineligibility (a list)
