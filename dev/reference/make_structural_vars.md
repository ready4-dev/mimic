# Make structural variables

make_structural_vars() is a Make function that creates a new R object.
Specifically, this function implements an algorithm to make structural
variables. The function returns Structural (a character vector).

## Usage

``` r
make_structural_vars(data_1L_chr = "Data", uid_1L_chr = character(0))
```

## Arguments

- data_1L_chr:

  Data (a character vector of length one), Default: 'Data'

- uid_1L_chr:

  Unique identifier (a character vector of length one), Default:
  character(0)

## Value

Structural (a character vector)
