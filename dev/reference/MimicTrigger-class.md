# MimicTrigger

Model event logic data.

## Slots

- `assert_1L_lgl`:

  Assert (a logical vector of length one)

- `event_1L_chr`:

  Event (a character vector of length one)

- `functions_ls`:

  Functions (a list)

- `use_1L_chr`:

  Use (a character vector of length one)

- `validate_chr`:

  Validate (a character vector)

- `x_MimicArguments`:

  (an instance of the MimicArguments class)

- `dissemination_1L_chr`:

  Dissemination (a character vector of length one)
